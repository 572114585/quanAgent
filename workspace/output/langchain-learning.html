Overview

什么是 LangChain？

一个用于构建大语言模型（LLM）应用的框架，提供模块化组件和编排能力。

LLM 应用的"操作系统"

LangChain 是一个开源框架，旨在简化和加速基于大语言模型的应用程序开发。它解决了 LLM 应用开发中的核心挑战——将不同的组件（模型、数据源、工具、记忆）编排成完整的应用。

无论你是构建 ChatBot、文档问答系统、自动化 Agent，还是复杂的 RAG 管道，LangChain 都提供了标准化的接口和丰富的集成生态，让你专注于业务逻辑而非底层胶水代码。

🔹 兼容 OpenAI、Anthropic、Google、开源模型等数十种 LLM
🔹 支持 100+ 文档加载器和向量数据库集成
🔹 活跃社区，GitHub 90k+ Stars

quickstart.py 
# LangChain 快速入门
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

# 1. 初始化模型
llm = ChatOpenAI(model="gpt-4")

# 2. 定义提示模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个AI助手"),
    ("human", "{input}")
])

# 3. 构建 Chain
chain = prompt | llm

# 4. 调用
result = chain.invoke({"input": "LangChain是什么？"})
print(result.content)
Core Concepts

六大核心概念

掌握这些核心抽象，你就能构建任何 LLM 应用。

⚡

Chains · 链

将多个组件串联成执行管道。支持 LCEL（LangChain Expression Language），用 | 操作符即可组合 Prompt → Model → Output Parser，简洁且可组合。

核心基础
🤖

Agents · 智能体

赋予 LLM 决策能力——让模型自主选择工具、规划步骤、执行动作。支持 ReAct、OpenAI Functions、Plan-and-Execute 等多种策略。

高级话题
🧠

Memory · 记忆

让对话拥有"记忆"——管理历史消息的存储与检索。支持 Buffer、Summary、Vector Store 等多种记忆类型，实现多轮对话上下文保持。

对话关键
📡

RAG · 检索增强生成

将外部知识注入 LLM：文档加载 → 文本分割 → 向量嵌入 → 相似检索 → 生成回答。有效解决 LLM 知识截止和幻觉问题。

企业必备
🔧

Tools · 工具

LLM 与外部世界交互的接口——搜索引擎、计算器、API 调用、数据库查询。定义工具并让 Agent 自主选择调用，极大扩展 LLM 能力边界。

能力扩展
🔗

LCEL · 表达式语言

LangChain 的统一链式编程范式。用管道运算符组合组件，原生支持流式输出、异步调用、并行执行、自动重试和配置追踪。

核心语法
Learning Path

五阶段学习路径

循序渐进，从入门到精通。

第一阶段 · 基础入门

了解 LangChain 设计理念，掌握环境搭建和基础 API 调用。学习 ChatPromptTemplate、ChatModel 和 StrOutputParser 三大基本组件。

环境配置 Prompt 模板 LLM 调用 Output Parser

第二阶段 · Chain 与 LCEL

深入学习 LCEL 表达式语言，掌握 Chain 的组合艺术。学习 Runnable 协议、并行分支、条件路由和自定义 Chain 的实现。

LCEL 语法 Runnable Chain 组合 流式输出

第三阶段 · RAG 检索增强

构建完整的 RAG 管道：文档加载、文本分割、向量化存储、语义检索。学习多种检索策略和 Advanced RAG 技术。

Document Loaders Text Splitters Vector Stores Embedding Models

第四阶段 · Agent 与 Tools

构建能自主决策的智能 Agent。学习 Tool 定义、Agent 类型、ReAct 循环、多 Agent 协作和复杂任务规划。

Tool 定义 Agent 类型 ReAct 循环 Toolkits

第五阶段 · 生产化与进阶

学习 LangSmith 可观测性、LangServe 部署、回调系统、缓存策略。掌握安全防护、性能优化和多模态应用的构建。

LangSmith LangServe Callbacks 多模态
